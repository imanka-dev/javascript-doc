Introduction

JavaScript is a versatile programming language used for web development.

It allows you to create dynamic and interactive experiences on websites.

console.log("Hello World!");
  • Client-side scripting
  • Event handling
Variables

Variables are containers for storing data values.

You can declare variables using var, let, or const.

let name = "John";
  • var - function scoped
  • let - block scoped
Functions

Functions are blocks of code designed to perform a task.

You can call functions to reuse code multiple times.

function greet() {
  return "Hello";
}
  • Function declaration
  • Function expression
Loops

Loops are used to repeat a block of code as long as a condition is met.

Common loops in JavaScript are for, while, and do...while loops.

for(let i=0; i<5; i++) {
  console.log(i);
}
  • For loop
  • While loop
Objects

Objects are collections of key-value pairs.

They are fundamental to JavaScript and used to model real-world entities.

let car = {
  brand: "Toyota",
  year: 2022
};

  • Methods
